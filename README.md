# 🏦 Banking System Project

This is a simple console-based banking system written in Python.

## ✅ Features
- Create new account
- Login to existing account
- Deposit money
- Withdraw money
- View transactions *(optional)*
- Data stored in `accounts.txt` and `transactions.txt`
